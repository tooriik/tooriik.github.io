Dual Booting

Dual Booting adalah proses menginstal dua sistem operasi, yaitu Windows dan Linux pada satu komputer. setiap sistem operasi diinstal pada partisi terpisah dan berjalan secara independen. Saat komputer dinyalakan sebuah boot loader seperti GRUB, akan menampilkan menu untuk memilih sistem operasi yang akan digunakan

Kali Linux

Menginstal Kali Linux didalam server industri. Kali Linux adalah distribusi Linux berbasis Debian yang dirancang khusus untuk kebutuhan keamanan komputer, seperti pengujian penetrasi (penetration testing), analisis keamanan, dan forensik digital.

Website VPS - Wordpress

Membangun Website Menggunakan VPS yang bisa diakses oleh online menggunakan IP VPS tersebut. WordPress adalah sebuah platform CMS berbasis open-source yang digunakan untuk membuat dan mengelola situs web. Dibangun dengan PHP dan menggunakan database MySQL atau MariaDB, WordPress awalnya dirancang sebagai alat untuk membuat blog, tetapi kini telah berkembang menjadi platform yang sangat fleksibel untuk membuat berbagai jenis situs web, seperti portofolio, toko online, hingga aplikasi web.

Botting dalam game

Melakukan Botting didalam game growtopia, yang membutuhkan berbagai kebutuhan IT seperti Proxy, IP Non Block HTTP, Socks5 ISP, Remote Desktop Control (RDP), Bypass, Script Pemrograman, Serta Email Farming. Botting adalah istilah yang merujuk pada penggunaan bot (program otomatis) untuk melakukan tugas-tugas tertentu secara otomatis tanpa perlu interaksi manusia secara langsung. Bot ini dirancang untuk menjalankan perintah tertentu secara efisien dan berulang-ulang, baik untuk tujuan yang sah maupun yang tidak sah.
